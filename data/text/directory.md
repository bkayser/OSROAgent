# Directory: Organizations, Leagues, Tournaments & Clubs That Assign and Pay Referees

This directory lists Oregon organizations (and similar entities) that assign and pay referees. Fill in missing assignor and PDF details from RefTown custom pages or organization websites.

**How to use:** Join the RefTown organization (or parent org for payor leagues), then set availability. Assignor and hotline info is needed for game-day issues and reporting.

---

## Oregon Youth Soccer (OYSA)

State body; games are assigned by **area assignors** (Bend, Gorge, Portland, Salem, Corvallis/Albany, Eugene, Roseburg, Southern Oregon, Longview/Vancouver WA). Referees are paid by direct deposit through RefTown.

| Field | Value |
|-------|--------|
| **Name** | Oregon Youth Soccer (OYSA) |
| **RefTown link** | https://reftown.com/default.asp?Assoc=NWSC--555 (OYSA games via NWSC and its payor leagues) |
| **Assignor** | Area assignors; list: https://reftown.com/link.asp?A=555&Q=RID%3D41&P=custom.asp (OYSA Assignors), https://reftown.com/link.asp?A=555&Q=RID%3D105&P=custom.asp (Rec Leagues) |
| **Assignor phone** | See RefTown assignor pages above for area assignor contacts |
| **Organization home page** | https://www.oregonyouthsoccer.org/ |
| **Rules PDF** | OYSA-Developmental-League-Rules-8.14.23.pdf (or current year equivalent on OYSA site) |

**Hotlines (statewide):** OYSA Referee Hotline (503) 332-5432; OYSA Game Day Hotline (971) 732-9897.
